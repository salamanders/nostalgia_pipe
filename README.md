# NostalgiaPipe

## **How It Works**

1.  **Discovery:** Point NostalgiaPipe at a hard drive containing raw `VIDEO_TS` folders. It recursively finds every DVD structure.
2.  **Analysis:** It identifies the content, snapping a high-quality reference image of the event.
3.  **Vision:** The AI analyzes the scene to generate a descriptive, sentimental title.
4.  **Preservation:** The video is transcoded, deinterlaced, and saved with rich metadata, ready for your media server or cloud library.
